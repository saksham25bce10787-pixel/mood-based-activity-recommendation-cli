# Mood-Based Activity Recommendation (CLI)

##  Overview
This project is developed as part of the AIML BYOP.
It is a CLI-based system that suggests activities based on the user's mood.  
The goal is to help students decide what to do according to their emotional state.

##  AI Concept Used
This project uses a **rule-based recommendation system**.  
It takes user input (mood) and provides suggestions using decision-making logic.

##  Features
- Mood selection (Happy, Sad, Stressed, Bored)
- Activity recommendations
- Input validation (handles incorrect input)
- Attempt limit (maximum 3 invalid attempts)
- Feedback system
- Loop for repeated use

## Technologies Used
- Python
- Command Line Interface (CLI)

## How to Run the Program
1. Make sure Python is installed on your system.
2. Open terminal or command prompt.
3. Navigate to the project folder.
4. Go inside the src folder:
   cd src
5. Run the program using:
   python main.py

## How to Test the Program
Follow these steps to test the system:

1. Enter your name when asked.
2. Select your mood by entering a number (1–4).
3. Check the suggested activities shown by the system.

## Test different cases:
- Enter valid choices (1–4) and observe correct output
- Enter invalid inputs (like letters or wrong numbers) to test validation
- Try entering wrong input 3 times to check attempt limit
- Give feedback (yes/no) and see system response
- Use the loop feature to test multiple moods

This will help verify that all features of the system are working correctly.
